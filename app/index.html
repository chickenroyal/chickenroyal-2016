Hi! I'm Andrew, a front-end developer from Newcastle.

I enjoy making responsive websites, using the latest web technologies. Some of my favourite buzzwords to use are: HTML5, CSS3 (especially when using SCSS), JavaScript (jQuery too!), SVG and Gulp.

Below is a selection of projects I've worked on. You can get in touch with me by email, or on Twitter.

vision2learn

Interactive e-learning courses for primary schools, secondary schools and adult learners.

Created using the responsive Foundation framework and SCSS to work on a wide range of devices. Courses included bespoke activities created with JavaScript/jQuery, HTML5 audio and video players, SVG graphics, and animations built using Adobe Edge Animate.

Brightwave

SCORM compliant e-learning courses for organisations.

Responsive courses created using the in-house Waveform engine. Activities include drag and drop quizzes, interactive videos, and scored multiple choice questions. Clients include Taylor Wimpey, Diageo, and Thames Water.

Rain or No Rain

Is it raining where you are? Of course there's a web-app for that.

Uses The Dark Sky Forecast and Google Maps APIs, as well as the HTML5 Geolocation interface.

Consult the Experts

Built using WordPress and a customised theme.
